Summary of the Basic Structure of an HTML Page

The basic structure of an HTML page is organized with the following elements:

  • <!DOCTYPE html>: Declares the document type, indicating that the page is written in HTML5.
  • <html lang="en">: The root element that wraps all content in the HTML document. The lang attribute specifies the primary language of the document (in this case, English).
  • <head>: Contains metadata about the page, such as character encoding, styles, scripts, and the page title.
  • <body>: Contains all the visible content of the page, including text, images, links, and more.

This structure is the essential foundation for creating any HTML page.
